Kiválasztás tétele

Adjuk meg, hogy egy számokból álló tömb, lista (iterálható adatszerkezet) hányadik helyen tartalmazza a 73-as számot.

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let index = 0;

for (let i = 0; i < tomb.length; i++) {

let feltetel = (tomb[i] === 73);

if (feltetel) {

index = i;

break;

}

}

console.log(`A 73-as számhoz tartozó index: ${index}`);

C#:

int[] tomb = { };

int index = 0;

for (int i = 0; i < tomb.Length; i++) {

bool feltetel = (tomb[i] == 73);

if (feltetel) {

index = i;

break;

}

}

Console.WriteLine($"A 73-as számhoz tartozó index: {index}");

Python:

tomb = []

index = 0

for i in range(len(tomb)):

feltetel = (tomb[i] == 73)

if (feltetel):

index = i

break

print(f"A 73-as számhoz tartozó index: {letezik}")
