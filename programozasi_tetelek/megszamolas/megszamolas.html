Megszámolás tétele

Adjuk meg egy számokból álló tömb, lista (iterálható adatszerkezet) elemeinek a számát.

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let darab = 0;

for (let i = 0; i < tomb.length; i++) {

darab++;

}

console.log(`A tömb elemeinek száma: ${darab}`);

C#:

int[] tomb = { };

int darab = 0;

for (int i = 0; i < tomb.Length; i++) {

darab++;

}

Console.WriteLine($"A tömb elemeinek száma: {darab}");

Python:

tomb = []

darab = 0

for i in range(len(tomb)):

darab += 1

print(f"A tömb elemeinek száma: {darab}")
