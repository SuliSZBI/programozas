Összegzés tétele 1. (Számok)

Adjuk meg egy számokból álló tömb, lista (iterálható adatszerkezet) elemeinek az összegét.

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let osszeg = 0;

for (let i = 0; i < tomb.length; i++) {

osszeg += tomb[i];

}

console.log(`A tömb elemeinek összege: ${osszeg}`);

C#:

int[] tomb = { };

int osszeg = 0;

for (int i = 0; i < tomb.Length; i++) {

osszeg += tomb[i];

}

Console.WriteLine($"A tömb elemeinek összege: {osszeg}");

Python:

tomb = []

osszeg = 0

for i in range(len(tomb)):

osszeg += tomb[i]

print(f"A tömb elemeinek összege: {osszeg}")
