Másolás tétele

Másoljuk át egy számokból álló tömb, lista (iterálható adatszerkezet) elemeinek kétszeresét egy másik tömbbe, listába (iterálható adatszerkezetbe).

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let masoltTomb = [];

for (let i = 0; i < tomb.length; i++) {

masoltTomb.push(2 * tomb[i]);

}

console.log(`A másolt tömb elemei: ${masoltTomb}`);

C#:

int[] tomb = { };

int[] masoltTomb = new int[tomb.Length];

for (int i = 0; i < tomb.Length; i++) {

masoltTomb[i] = 2 * tomb[i];

}

Console.WriteLine("A másolt tömb elemei: ");

masoltTomb.ToList().ForEach(elem => Console.WriteLine(elem));

Python:

tomb = []

masoltTomb = []

for i in range(len(tomb)):

masoltTomb.append(2 * tomb[i])

print(f"A másolt tömb elemei: {masoltTomb}")
