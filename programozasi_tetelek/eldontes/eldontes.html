Eldöntés tétele

Adjuk meg, hogy egy számokból álló tömb, lista (iterálható adatszerkezet) tartalmazza-e a 73-as számot.

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let letezik = false;

for (let i = 0; i < tomb.length; i++) {

let feltetel = (tomb[i] === 73);

if (feltetel) {

letezik = true;

break;

}

}

console.log(`A tömb tartalmazza a 73-as számot: ${letezik}`);

C#:

int[] tomb = { };

bool letezik = false;

for (int i = 0; i < tomb.Length; i++) {

bool feltetel = (tomb[i] == 73);

if (feltetel) {

letezik = true;

break;

}

}

Console.WriteLine($"A tömb tartalmazza a 73-as számot: {letezik}");

Python:

tomb = []

letezik = False

for i in range(len(tomb)):

feltetel = (tomb[i] == 73)

if (feltetel):

letezik = True

break

print(f"A tömb tartalmazza a 73-as számot: {letezik}")
