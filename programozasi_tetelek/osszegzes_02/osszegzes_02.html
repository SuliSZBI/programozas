Összegzés tétele 2. (Sztringek)

Adjuk meg egy sztringekből álló tömb, lista (iterálható objektum) elemeiből képzett mondatot.

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let mondat = '';

for (let i = 0; i < tomb.length; i++) {

mondat += `${tomb[i]} `;

}

console.log(`A tömb elemeiből képzett mondat: ${mondat}`);

C#:

string[] tomb = { };

string mondat = "";

for (int i = 0; i < tomb.Length; i++) {

mondat += $"{tomb[i]} ";

}

Console.WriteLine($"A tömb elemeiből képzett mondat: {mondat}");

Python:

tomb = []

mondat = ""

for i in range(len(tomb)):

mondat += f"{tomb[i]} "

print(f"A tömb elemeiből képzett mondat: {mondat}")
