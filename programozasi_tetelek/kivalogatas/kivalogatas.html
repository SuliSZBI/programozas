Kiválogatás tétele

Válogassuk ki egy számokból álló tömb, lista (iterálható adatszerkezet) elemei közül azokat egy másik tömbbe, listába (iterálható adatszerkezetbe), amelyek hárommal oszhatóak.

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let harommalOszthato = [];

for (let i = 0; i < tomb.length; i++) {

let feltetel = (tomb[i] % 3 === 0);

if (feltetel) {

harommalOszthato.push(tomb[i]);

}

}

console.log(`A kiválogatott tömb elemei: ${harommalOszthato}`);

C#:

int[] tomb = { };

List<int> harommalOszthato = new List<int>();

for (int i = 0; i < tomb.Length; i++) {

bool feltetel = (tomb[i] % 3 == 0);

if (feltetel) {

harommalOszthato.Add(tomb[i]);

}

}

Console.WriteLine("A kiválogatott tömb elemei: ");

harommalOszthato.ForEach(elem => Console.WriteLine(elem));

Python:

tomb = []

harommalOszthato = []

for i in range(len(tomb)):

feltetel = (tomb[i] % 3 == 0)

if (feltetel):

harommalOszthato.append(tomb[i])

print(f"A kiválogatott tömb elemei: {harommalOszthato}")
