Szétválogatás tétele

Válogassuk szét egy számokból álló tömb, lista (iterálható adatszerkezet) elemeit páros és páratlan számokra. Tároljuk ezeket külön-külön adatszerkezetben!

Megoldás programozási nyelveken:

JavaScript:

let tomb = [];

let paros = [];

let paratlan = [];

for (let i = 0; i < tomb.length; i++) {

let feltetel = (tomb[i] % 2 === 0);

if (feltetel) {

paros.push(tomb[i]);

} else {

paratlan.push(tomb[i]);

}

}

console.log(`A tömb páros elemei: ${paros}`);

console.log(`A tömb páratlan elemei: ${paratlan}`);

C#:

int[] tomb = { };

List<int> paros = new List<int>();

List<int> paratlan = new List<int>();

for (int i = 0; i < tomb.Length; i++) {

bool feltetel = (tomb[i] % 2 == 0);

if (feltetel) {

paros.Add(tomb[i]);

} else {

paratlan.Add(tomb[i]);

}

}

Console.WriteLine($"A tömb páros elemei: ");

paros.ForEach(elem => Console.WriteLine(elem));

Console.WriteLine($"A tömb páratlan elemei: ");

paratlan.ForEach(elem => Console.WriteLine(elem));

Python:

tomb = []

paros = []

paratlan = []

for i in range(len(tomb)):

feltetel = (tomb[i] % 2 == 0)

if (feltetel):

paros.append(tomb[i])

else:

paratlan.append(tomb[i])

print(f"A tömb páros elemei: {paros}")

print(f"A tömb páratlan elemei: {paratlan}")
